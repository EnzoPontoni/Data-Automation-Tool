<br/>

- **Arquitetura Segura no Electron**: Utiliza `preload.js` nativo e `ipcMain/ipcRenderer` (Context Bridge) para bloquear o acesso do contexto do browser ao Node.js, respeitando as práticas de segurança *(Context Isolation e Node Integration disabled)*.
- **Processamento Dinâmico de Regex**: Analisa e processa automaticamente milhares de linhas de `.txt` via Expressões Regulares (Regex) para extrair estruturas de objetos ou chaves únicas.
- **Deduplicação e Manipulação de Streams**: Limpa duplicatas de arquivos baseadas em UUIDs e utiliza data streams nativos para gerar arquivos compactados (`.zip`) eficientes em memória com a biblioteca `archiver`.
- **Comunicação de Rede Avançada**: Manipula uploads em massa na API Gofile usando `axios` com requisições limitadas configuradas e buffers gerados programaticamente com `form-data`.
- **UI Moderna Personalizada**: Interface Glassmorphism (efeito vidro difuso com transparências e blurs customizados) desenhada 100% em Vanilla CSS com layout multi-fásico e barra superior (`AppRegion Drag`) original do Electron.

Para rodar este projeto na sua máquina de desenvolvimento localmente:

```bash
# Clone o repositório
git clone https://github.com/EnzoPontoni/data-automation-tool.git

# Entre na pasta
cd data-automation-tool

# Instale as dependências locais
npm install

# Execute a build de desenvolvimento do Electron
npm start
```

```bash
# Para compilar o instalador distribuível (.exe)
npm run build
```
