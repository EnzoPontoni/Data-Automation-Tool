<br/><br/>

Uma ferramenta Desktop simples e eficiente construída para automatizar tarefas de processamento de dados. Desenvolvi esse app para tratar dados em lotes (lendo e extraindo de `.txt`), compilar automaticamente tudo em arquivos `.zip` e gerenciar uploads pesados para a nuvem de forma nativa e rápida.
